## What This Does

BoardGameArena (BGA) has hundreds of games, but many require a Premium subscription to play. Finding which games are actually free to play is tedious — the free games are mixed in with premium ones on the game list, and there's no "show only free" filter.

This project solves that by maintaining a curated list of known-free games, displaying them in a clean dashboard with:

- **Search** — find games by name
- **Genre filter** — narrow down by Family, Card, Strategy, Abstract, Thematic, or Dice
- **Sort options** — by name, popularity, or player count
- **Popularity bars** — relative play-frequency estimates based on BGA's actual play counts (log-normalized)
- **RSS feed** — subscribe to get notified when new free games are discovered
- **Auto-refresh** — the server re-checks every 60 seconds
- **Dark mode** — toggle stored in localStorage, persists across visits

The theme is deliberately **boxy** (no rounded corners) with a dark beige / burgundy color palette.

## How the Game List Is Maintained

Each game in the list was **manually verified** as free by visiting:

```
https://boardgamearena.com/gamepanel?game=SLUG
```

and confirming the "Free Game" badge. When a game is verified as free, it's added to `scraper.js` in the `BASE_KNOWN_GAMES` array with its BGA slug and player count.

When the server starts, it attempts to scrape BGA's game list pages for any new free games. Since BGA blocks automated requests (returns HTTP 302/404), this almost always falls back to the curated list. The scraping attempt is kept as a best-effort for when BGA's defenses are relaxed.

### Genre Assignment
Genres are assigned via keyword matching against the game name. For example, games containing "chess", "checkers", or "hive" get the Abstract genre. If no keywords match, the game defaults to Family.

### Popularity Scores
Popularity is derived from BGA's "Number of games played" statistic shown on each game's page. Raw play counts are normalized using a logarithmic scale:

```
score = log10(playCount) / log10(maxPlayCount) × 100
```

Where Yahtzee (17.3 million plays) is the max. This gives a score between 0–100 that reflects relative popularity without being skewed by the huge range of play counts.

## Tech Stack

- **Backend:** Node.js, Express
